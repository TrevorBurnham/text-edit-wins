Emmet

HTML Editing Zen

What is Emmet?

Formerly known as Zen Coding, Emmet is a Swiss Army knife for writing and editing HTML. Its signature feature is expanding CSS selectors into markup. For example, if you type "p" and then Tab, Emmet will give you a matching pair of paragraph tags. If you type "p.story", Emmet will give you those tags with the class="story" attribute.

Advanced markup abbreviations

Emmet has a notion of addition and multiplication. Want to create an unordered list with three children? Just type "ul>li*3", then Tab. Try it! You can even Tab between the fields you create that way.

You can find more examples at docs.emmet.io/css-abbreviations/.

CSS abbreviations

In addition to expanding CSS selectors into HTML, Emmet also gives you shorthands for properties in CSS. For example, rather than typing border-radius: 8px, you can just type br8 and then hit Tab.

You can find more examples at docs.emmet.io/abbreviations/.

Commands

As if expanding abbreviations weren't enough, Emmet comes with a variety of commands (available in Sublime via the Command Palette) for editing HTML. Among the most useful:

  • Wrap with Abbreviation: Want to put some text in a span with class="icon"? Just select it, use this command, and enter "span.icon".
  • Remove Tag: Oops, don't need that span after all? Kill it in one keystroke.
  • Select Next/Previous Item: Select the next opening tag, or the next attribute within a tag.

You can find more examples at docs.emmet.io/actions/.
